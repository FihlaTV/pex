• HTMLAudio.js

  • ¶

    Simple audio playback for the browser

    Example use

     var audio = new Audio("assets/test.mp3");
 audio.play();
 audio.volume = 0.5;

    Reference

  • ¶

    Audio(file)

    file - audio file to play

    function Audio(url) {
  this.audio = new window.Audio();
  this.audio.src = url;

  this.audio.onended = function() {
    this.playing = false;
  }.bind(this);
